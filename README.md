# 🚀 Developer Productivity Predictor

## 📌 Overview

Developer Productivity Predictor is a Machine Learning web application that predicts a software developer's productivity based on coding activity, AI tool usage, commits, bugs reported, sleep hours, distractions, cognitive load, and stress level.

The application uses an AdaBoost Classifier model trained on developer productivity data and provides instant predictions through an interactive web interface built with Flask.

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Developer-Productivity-Predictor.git
cd Developer-Productivity-Predictor
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
